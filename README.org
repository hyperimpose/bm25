* bm25: An Okapi BM25 implementation in pure Erlang

This is a pure Erlang library that implements the Okapi BM25 algorithm.

Features:
- Inverted Index
- Customizable tokenization function

** Usage

#+BEGIN_SRC erlang
  %% You provide your own tokenization function to clean-up and split text into tokens.
  Index1 = bm25:init(#{tokenize => fun tokenize/1}),

  %% Add documents // Each document must have a unique Id.
  Index2 = bm25:add(Index1, Id1, Doc1)
  Index3 = bm25:add(Index2, Id2, Doc2)

  bm25:search(Index3, "query"),
#+END_SRC
  
** Build

**** rebar3
~bm25~ is built using rebar3. You can compile it by running: ~$ rebar3 compile~.

or you can include it in your projects as a dependency in rebar.config:
#+BEGIN_SRC erlang
  {deps, [%% ... Other dependencies
          {bm25, {git, "https://github.com/hyperimpose/bm25.git", {branch, "master"}}}]}.
#+END_SRC
